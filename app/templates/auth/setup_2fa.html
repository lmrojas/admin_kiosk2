Configurar Autenticación de Dos Factores

Sigue los pasos para activar la autenticación de dos factores

Paso 1: Escanea el código QR

Usa una aplicación de autenticación como Google Authenticator o Authy para escanear este código QR

Paso 2: Ingresa el código

Ingresa el código de 6 dígitos que aparece en tu aplicación de autenticación

Información importante
  • La autenticación de dos factores agrega una capa adicional de seguridad a tu cuenta.
  • Necesitarás ingresar un código cada vez que inicies sesión.
  • Guarda los códigos de respaldo en un lugar seguro en caso de perder acceso a tu dispositivo.
  • Puedes desactivar 2FA en cualquier momento desde la configuración de tu cuenta.
